## The Drake

> Simple chess-like game in Java

___
### Technologies:
  * JavaFX
  * Java 11
  * maven

___
### Install and Run

#### Requirements:
  * maven 3.8.5 (can be downloaded and installed [here](https://maven.apache.org/download.cgi))
  * java 11+

#### compiling:
```bash
mvn compile
```

#### running:
```bash
mvn exec:java -Dexec.mainClass=com.example.thedrake.ui.TheDrakeApp
```

#### cleaning:
```bash
mvn clean
```
